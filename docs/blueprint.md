# **App Name**: LineCommand

## Core Features:

- Secure User Authentication: Implement a secure login system for two distinct roles: Admin and Employee, powered by Firebase Authentication.
- Employee Data Management (Admin): Admins can perform CRUD operations (Add, Edit, Delete) on employee profiles, including PNO, name, and other essential details, storing this data in Firebase.
- Duty Assignment and Viewing: Admins can assign daily duties to employees. Employees can view their assigned daily duties and review their duty history with custom date range filtering from Firebase.
- Leave Management: Admins can update and manage employee leave requests and details. Employees can view their leave status and past leave history in a dedicated section.
- Search and Filter Capabilities: Admins can efficiently search for employees by PNO and filter both duty and leave records by a specified date range.
- PDF Report Generation: Enable users (Admins and Employees) to generate and download formatted PDF reports of duty assignments and leave details.
- Intuitive Bottom Navigation: Provide clear and consistent navigation with a bottom bar featuring 'Dashboard', 'Duty', 'Leave', and 'Profile' sections for easy access.

## Style Guidelines:

- Primary color: A sophisticated, muted purple (#9970C2) to evoke professionalism and modernity. This will be used for interactive elements and key headings.
- Background color: A very light, almost white, with a subtle purple tint (#F2F0F4), providing a clean and crisp canvas, enhancing readability.
- Accent color: A deep, professional indigo blue (#3D3D8F), offering strong contrast for call-to-action buttons and important alerts, while complementing the primary purple.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics suitable for data display and professional content.
- Use a set of clear, contemporary line-art icons that are consistent in style, ensuring quick recognition for duties, leaves, profile, and administrative actions.
- Implement a clean and structured layout with ample whitespace. The core navigation will utilize a bottom bar with 'Dashboard', 'Duty', 'Leave', and 'Profile' tabs for quick access.
- Incorporate subtle and smooth transition animations between screens and upon data loading to provide a polished and responsive user experience.
